Rock-Paper-Scissors is a simple console game. The main purpose of this project was to create a multi-threaded and multi-transactional server that can handle multiple game instances and connections at once. I implemented this using C++ and Socket Programming with the text editor Notepad++. The server is meant to run on a UNIX OS while the client can run on either Windows or UNIX.
