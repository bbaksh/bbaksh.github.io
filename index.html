My Latest Interests.

I am always interested in learning new things!
Here are the latest things that are on my mind...

Machine Learning.

Machine learning is a subfield of computer science that evolved from the study of pattern recognition and computational learning theory in artificial intelligence. Machine learning explores the study and construction of algorithms that can learn from and make predictions on data. Currently, I am enrolled in a coursera online course hosted by Andrew Ng to learn Machine Learning outside of university as a personal interest.

Neural Networks.

A Neural network is an information processing system that is inspired by the way biological nervous systems, such as the brain, process information. It is composed of a large number of highly interconnected processing elements (neurones). Neural networks, like people, learn by example. Recently, I found a youtube video called Mar/IO that intrigued me. A computer program utilized a neural network to learn how to play and complete the popular video game, Super Mario Bros.

Android Development.

Android software development is the process by which new applications are created for the Android operating system. Applications are usually developed in Java programming language using the Android software development kit (SDK), but other development environments are also available. I have taken a great interest in mobile development, specifically android, and I am currently working towards creating games and applications as side projects to learn.

Cloud Computing.

Cloud computing is the practice of using a network of remote servers hosted on the Internet to store, manage, and process data, rather than a local server or a personal computer. This is where the software industry is headed in my opinion. Everything one day will be hosted on the cloud and offered through the popular distribution method of "Software as a Service" (SaaS). It amazes me that one day we will be able to access any piece of infromation or application through any device becuse of cloud computing. In school and outside of school, I try to stay up to date with the big changes that are happening.

Augmented Reality.

Augmented reality is a live direct or indirect view of a physical, real-world environment whose elements are augmented by computer-generated sensory input such as sound, video, graphics or GPS data. As a result, the technology functions by enhancing one’s current perception of reality. Currently, I have taken an interest in learning about how AR works. Soon I will be creating a small application to scratch the surface of AR and learn more about it.

Web Development.

Web development is a broad term for the work involved in developing a web site for the Internet (World Wide Web) or an intranet (a private network). With the rapid expansion of cloud computing; web development is becoming a must-have for developers looking to stay up-to-date with the skewing of the industry. I have completed my Web Technologies course in school, and I learned that I actually love Web Development. I have taken an interest in Front-End and Back-End development using web languages and I am currently working toward honing my technical skills in this area.
